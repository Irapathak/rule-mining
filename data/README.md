# Data directory

For datasets other than the default **acs** (which lives in `../acs/`), place files here so paths in `configs/config.json` resolve inside the repo:

- **german:** `data/german_credit/german_data_not_encoded.csv`
- **so:** `data/stackoverflow/so_countries_col_new.csv`

Rules files (e.g. `GermanChosen10Treatments.json`, `Chosen10Treatments.json`) live in `algorithms/` and are already in the repo.
